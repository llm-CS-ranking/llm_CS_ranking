# Generated Result Artifacts

This directory contains lightweight, publication-facing outputs:

- `tables/`: CSV and LaTeX tables used by the paper and appendix.
- `common_window/`: common-calendar robustness tables and transaction-cost sensitivity outputs.

Raw LLM rank-panel pickles and API metadata are intentionally not included in the public release. The included CSV files are sufficient to re-render the LaTeX tables and selected factor-loading figures.
